# The Ethical Hacker's Command Handbook

A static, GitHub Pages-ready handbook for authorised security learning.

## Publish with GitHub Pages

1. Copy this folder's contents into the root of a GitHub repository.
2. Open **Settings → Pages** in GitHub.
3. Select **Deploy from a branch**, choose `main` and `/(root)`, then save.

## Add chapters

Duplicate a chapter page, update the header links, and add a matching card to `index.html`.

Keep all examples limited to systems you own or have written permission to test.
